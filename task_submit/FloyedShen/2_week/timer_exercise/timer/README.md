###  TIMER

- **作者**：申国斌（Floyed_Shen）
- **修订**：Floyed_Shen
- **日期**：2018-10-4
- **版本**：1.0.0
- **摘要**：这是一个简单的计时器程序，在显示时间的同时接收用户命令，命令包括start、end、pause和resume

- **Functions**:
###void open();
    在程序中调用此函数、开始运行计时器

- **Depend Libraries**:
    需要包括opencv2
    请使用兼容c++11标准的编译器

### 编译提示:
    cmake .
    make
    bin/./TIMER_PRO

### How to run:
    cd bin
    ./TIMER_PRO
### the result:
    参见./result文件夹
