## 如何使用

- 在终端中**输入**初始化秒数（以毫秒计），按下 <**Enter**> 即可开始。

- 刷新频率为1s.

- 若要停止，在终端中输入**STOP**即可停止计时器的运作。

- 若要退出，在终端中输入**QUIT**即可退出计时器。
