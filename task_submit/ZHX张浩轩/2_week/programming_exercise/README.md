- Version: 1.0.0
- Information: This is my IP Network programme.  

### How to run?
	First: g++ IP.cpp -o IP.out  
	Then: ./IP.out  

# How to input?
	Input the total number first, then input the ip one by one;  
	Output will appear after input.  
	
	For example:  

	Sample Input:  
	3  
	194.85.160.177  
	194.85.160.183  
	194.85.160.178  
	
	Sample Output:  
	194.85.160.176  
	255.255.255.248
