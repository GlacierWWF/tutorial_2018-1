### how to use
1. 创建自己的数据集

            `python gesture.py` <br>
            press `2`   `s`  录制手势图片<br>
2. 训练数据集

             `python gesture.py`
             press `1`
             开始训练
3. 测试

             `python gesture.py`
             press `2`
             开始预测时,press `p` 
        >>>     在屏幕显示出预测结果
